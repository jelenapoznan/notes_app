main.py - file that we're going to run when we want to start our web server
__init__.py - this file is going to make website folder a python package (we can import that folder, and whatever is insede of __init__.py file will run automatically. Here we are going to setup flask application)
models.py - we use to store database models
views.py - stores all the main views or the URL endpoints for the functioning/frontend aspect of our website
flask modules: flask-login (is for login users in), flask-sqlalchemy (wrapper for SQL, to create database models, delate models, add models)
